## What This Means

### If Accurate: Major Strategic Insight

**China's semiconductor "self-sufficiency" push did NOT translate to patent growth**

Possible explanations:
1. **Manufacturing focus over innovation** - Building fabs, not inventing new tech
2. **Technology import substitution** - Using existing foreign IP domestically
3. **Trade secrets over patents** - Protecting innovations as confidential, not filing
4. **Quality over quantity shift** - Fewer but higher-value semiconductor patents
5. **Patent classification shift** - Semiconductor innovations now filed under AI, telecom, etc.

### If Data Issue: Methodological Problem

**CPC H01L might be too narrow or outdated**

Semiconductor-related patents might now be filed under:
- **G06N** (AI chips, neural processors)
- **H04L** (5G chips, telecommunications hardware)
- **H03K** (Digital circuits)
- **G11C** (Memory devices)
- **B82Y** (Nanotechnology for semiconductors)

## Hypothesis: Patent Classification Drift

### The "AI Chip" Problem

Modern semiconductor patents may be classified as:
- **AI/ML chips** → G06N (not H01L)

- **IoT processors** → H04W (not H01L)
- **Quantum processors** → G06N10 (not H01L)

**If semiconductors now appear in multiple CPC codes, H01L-only analysis would miss them.**

## DATA QUALITY ASSESSMENT

### Systematic Issues Identified:

**Problem:** Extreme year-to-year volatility across ALL three technologies

**Evidence:**
- cs.CL (AI): 25 → 4,000 → 25 papers (2020-2022)
- physics.atom-ph (Quantum): 800 → 1,224 → 100 → 1,303 → 1 papers
- astro-ph.EP (Space): 200 → 1,600 → 100 → 25 papers

**Root Causes:**
1. **API query construction:** Date range formatting may be incorrect
2. **Result pagination:** 2K limit per query, may miss papers beyond limit
3. **Category changes:** arXiv may have reclassified papers between categories

**Impact:**
- **Absolute counts unreliable** (year-to-year changes >100% suspect)
- **Growth trends directional only** (CAGR calculations valid IF endpoints correct)
- **Category totals more reliable** than year-by-year breakdowns

**Mitigation:**
- ✅ Transparently documented anomalies in all summaries
- ✅ Triangulated with other sources (CORDIS, OpenAlex, market data)
- ✅ Used multiple data points (not relying on arXiv alone)
- ⏭️ **Next:** Retry with Kaggle dataset (1.7M papers, offline, complete)

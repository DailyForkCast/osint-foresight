# Comprehensive OSINT Data Sources for Technology & Research Intelligence
**Last Updated:** 2025-09-21
**Purpose:** Strategic intelligence gathering on international research collaborations and technology transfer

## 1. Government Procurement & Contracting Databases

### United States
- **SAM.gov** (System for Award Management)
  - URL: https://sam.gov
  - Coverage: All US federal contracts, grants, assistance listings
  - API: Available with registration
  - Key data: Contractor details, award amounts, performance locations

- **USAspending.gov**
  - URL: https://www.usaspending.gov
  - Coverage: All US federal spending
  - API: Bulk download and REST API
  - Key data: Contract awards, grants, loans, direct payments

- **FPDS-NG** (Federal Procurement Data System)
  - URL: https://www.fpds.gov
  - Coverage: Federal contract actions over $10,000
  - API: ATOM feeds and bulk downloads
  - Key data: Detailed contract modifications, vendor information

### European Union
- **TED (Tenders Electronic Daily)**
  - URL: https://ted.europa.eu
  - Coverage: EU public procurement above thresholds
  - API: Open Data Portal with bulk XML/CSV
  - Key data: €750+ billion annual procurement

- **EU Open Data Portal**
  - URL: https://data.europa.eu
  - Coverage: All EU institutions data
  - API: SPARQL endpoint, REST APIs
  - Key data: Research funding, procurement, statistics

- **CORDIS**
  - URL: https://cordis.europa.eu
  - Coverage: EU research projects and results
  - API: RESTful API with JSON/XML
  - Key data: Horizon 2020, FP7, partner networks

## Data Collection Best Practices

1. **API Rate Limiting**
   - Implement exponential backoff
   - Use bulk downloads when available
   - Cache responses locally

2. **Data Validation**
   - Cross-reference multiple sources
   - Maintain provenance tracking
   - Version control data schemas

3. **Legal Compliance**
   - Respect robots.txt
   - Follow terms of service
   - Maintain attribution

4. **Storage Strategy**
   - Use columnar formats (Parquet) for analytics
   - Implement data partitioning by date/country
   - Regular incremental updates

- [ ] Personnel exchange confirmed

## ðŸ” CORROBORATION REQUIREMENTS

### Triple Source Rule for Critical Claims
1. **Primary Source**: Original document/statement
2. **Secondary Confirmation**: Independent verification
3. **Tertiary Check**: Different source type

Example:
- Primary: Leonardo annual report states 40+ helicopters in China
- Secondary: Chinese aviation registry shows registrations
- Tertiary: Trade publication reports on deliveries

### Negative Evidence Search
- [ ] Searched for contradicting information
- [ ] Checked for retractions/corrections
- [ ] Looked for industry disputes about claims
- [ ] Verified no classification issues
- [ ] Confirmed information is public domain

## âš ï¸ RED FLAGS REQUIRING EXTRA SCRUTINY

### Information Quality Issues
- [ ] Single source only
- [ ] Anonymous sources
- [ ] Outdated information (>3 years)
- [ ] Conflicting reports exist
- [ ] Information seems "too good"
- [ ] Timing seems coincidental

### Logical Consistency Issues
- [ ] Violates known regulations
- [ ] Contradicts established patterns
- [ ] Requires multiple failures
- [ ] No one else reporting
- [ ] Seems economically irrational
- [ ] Technology timeline doesn't match

## ðŸ“‹ VERIFICATION WORKFLOW

### Phase 1: Initial Discovery
```
1. Document exact finding
2. Note source and date
3. Screenshot/archive evidence
4. Record initial assessment
```

### Phase 2: Rapid Verification (1-2 hours)
```
1. Check entity names/spellings
2. Verify basic facts
3. Search for corroboration
4. Check for contradictions
```

### Phase 3: Deep Verification (4-8 hours)
```
1. Triple source critical claims
2. Create evidence matrix
3. Document access levels
4. Map exploitation pathways
```

### Phase 4: Alternative Analysis (2-4 hours)
```
1. Generate alternative explanations
2. Test each alternative
3. Document why rejected/accepted
4. Calculate probability bands
```

### Phase 5: Peer Review (If Available)
```
1. Present to colleague
2. Address challenges
3. Refine analysis
4. Document disagreements
```

2. **ALWAYS** 30 in upper band [30,60)
3. **ALWAYS** include critical findings (even incomplete)
4. **ALWAYS** check current date
5. **ALWAYS** apply Arctic override
6. **ALWAYS** use zero-trust fusion
7. **ALWAYS** weight by value not count

### Common Mistakes to Avoid:

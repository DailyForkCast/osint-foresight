## üìö UNIVERSAL CITATION STANDARD

### In-Text Citation Format
Use lowercase Roman numerals in parentheses at the end of the relevant sentence or clause:
- Single citation: `(i)`
- Multiple citations: `(ii)(iii)` or `(iv, v, vi)` for same point
- Range: `(vii-ix)` for continuous sources

### Endnote Format Structure
```
(i) [TYPE] Title/Description. Author/Organization. URL. Date Published. Accessed: YYYY-MM-DD. Archive: [URL if critical]
```

### Source Type Tags
- **[WEB]** - General web sources
- **[GOV]** - Government (.gov, .mil, official ministry sites)
- **[CORP]** - Corporate (company reports, press releases)
- **[ACAD]** - Academic (journals, university research)
- **[NGO]** - Non-governmental organizations
- **[MEDIA]** - News media
- **[REG]** - Registries/filings (USPTO, Companies House, etc.)
- **[CC]** - Claude Code artifact (include file path)
- **[INT]** - International organizations (UN, NATO, EU)
- **[CONF]** - Conference proceedings/presentations

### Critical Claims Requiring Archives
- China connection allegations
- Financial/investment data
- Personnel movements
- Technology transfer claims
- Sanctions/controls violations
- Classified/sensitive program details

3. **Verify each URL** is accessible

5. **Check source dates** for currency
6. **Add source type tags** [GOV], [CORP], etc.
7. **Review for completeness** using checklist
8. **Number check** - ensure continuous and no orphans

```

```

### When Data Is Estimated
```

```

```

```

## üéØ UNIVERSAL OUTPUT CONTRACT (All Phases)

### Core Principle: **Narrative First, Data Second, Validation Always**

### Validation Requirements (ALL Phases)
Every finding must pass through validation gates:
1. **Evidence Verification** - Triple source rule for critical claims
2. **Alternative Testing** - 5+ alternatives considered
3. **Technology Assessment** - TRL and strategic value scored
4. **Oversight Gap Analysis** - How did this happen?
5. **Confidence Scoring** - 0-20 scale with explicit uncertainties

For EVERY phase response, deliver in this order:

1. **Analyst Narrative** (PRIMARY)
   - 600-1,000 words per major section
   - Clear topic sentences
   - Evidence-based assertions with inline citations (i), (ii), etc.
   - Probability bands inline: *(Probability: 30-60%; Confidence: Med)*

2. **Key Judgments**
   - 3-7 bullets maximum
   - One sentence each, no hedging
   - Include probability/confidence for forward-looking statements

3. **What It Means**
   - 1-3 paragraphs
   - Policy/operational/industry implications
   - Decision points and coordination needs

4. **Evidence Endnotes**
   - Numbered with roman numerals: (i), (ii), (iii), etc.
   - Format: `(i) [Source Type] Title. Organization. URL. Date Published. Accessed: YYYY-MM-DD. Archive: [URL if critical]`
   - Source type tags: **[WEB]**, **[GOV]**, **[CORP]**, **[ACAD]**, **[CC]** (Claude artifact), **[REG]** (registry/filing)

5. **Data Artifacts** (APPENDIX)
   - JSON/CSV blocks
   - Tables and matrices
   - Technical details

### Citation Format Standard
```
In-text: "Leonardo maintains a Beijing office for helicopter sales (i)."
Endnote: "(i) [CORP] Annual Report 2023. Leonardo S.p.A. https://leonardo.com/annual-report-2023. 2024-03-15. Accessed: 2025-09-14."
```

### Token Management Rules
- **NEVER** cut the Analyst Narrative
- Trim in this order: Appendix depth ‚Üí Endnote details ‚Üí Key Judgments
- Maintain all critical citations even under pressure

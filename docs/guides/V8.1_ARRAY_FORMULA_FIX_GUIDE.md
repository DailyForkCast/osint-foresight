## üö® COMMON MISTAKES

### **Mistake 1: Typing the braces**
‚ùå DON'T type `{` and `}` manually
‚úÖ Excel adds them automatically when you press Ctrl+Shift+Enter

### **Mistake 2: Pressing just Enter**
‚ùå Regular Enter won't make it an array formula
‚úÖ Must press Ctrl+Shift+Enter

### **Mistake 3: Editing the formula**
‚ùå If you edit and press Enter, braces disappear
‚úÖ Always finish edits with Ctrl+Shift+Enter

## üìö WHAT ARE ARRAY FORMULAS?

**Simple explanation:**
- Regular formula: Looks at one value, returns one value
- Array formula: Looks at multiple values, does calculations, returns one value
- Needs special entry: Ctrl+Shift+Enter to tell Excel "this is an array formula"

**Example in our tracker:**
```
Regular formula: =INDEX(Table[Column], 2)
  ‚Üí Gets the value from row 2

Array formula: =INDEX(Table[Column], SMALL(IF(Table[ID]="PRJ-001", ROW(Table[ID])), 1))
  ‚Üí Looks at ALL rows, finds ones matching PRJ-001, gets the 1st match
  ‚Üí Needs Ctrl+Shift+Enter to work
```
